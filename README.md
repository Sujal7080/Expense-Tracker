# Expense Tracker with Data Visualization

A responsive personal finance tracker built using JavaScript, Chart.js, and Local Storage. The app allows users to add, edit, and delete transactions while visualizing spending patterns through dynamic charts. Designed with a clean dashboard layout using Flexbox and CSS Grid, providing a simplified overview of expenses, categories, and budget status.
